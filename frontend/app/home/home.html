Frequently Asked Questions

  1. What is the current status of the website?

    • We are currently in a open alpha version so your feedback and bug reports will help tremendously!

  2. How does your service work?

    1. You signup and determine which games you would like to track.
    2. When adding a game you give us the identifying information we need to get the stats from the game.
    3. Every hour* we reach out to the game provider and get updated stats.
    4. We then store those stats in our database for future reference.

  3. Can I request for a game to be added?

    • Yes. Submit a contact request below and we will take a look at it. If possible, please include a link to the game's API.

  4. Will there be a premium service?

    • This is currently up in the air depending on how popular the website gets.
    • We were thinking services like on demand updates, graphing over time and such would be great premium features.
    • Have a feature you want? Please let us know by using the contact form below!
