Dashy

Files

Dashy Angular Theme consists of AngularJS view files.
  • The Directives are located in app/scripts/directives/
  • The view files are located in app/views/
  • The controllers are located in app/scripts/controllers/
  • The app.js file is located in app/scripts/
  • There are specific less files for separate widgets and pages which are located in app/styles/less/widgets and app/styles/less/pages respectively.

Theme Colors

Currently there are 3 different theme colors for Dashy Angular Theme, other than the default one. You can choose each one from the Color Select button on right side of the navigation bar. To select a theme color, you have to include the theme specific .css file in your index.html. The available theme colors are

  • app-green.css (Default)
  • app-red.css
  • app-blue.css
  • app-orange.css
